Modified & Updated version of [Screen Grabber 4.2.0](https://github.com/SymbianSource/oss.FCL.sf.tools.rndtools/tree/master/screengrabber)

an application for (Symbian S60v3 and above) that can be used for
recording the screen or taking screenshoots.
it supports various formats.

this version uses [msf_gif](https://github.com/notnullnotvoid/msf_gif) for exporting the screen recording as high-quality GIF Animation.
