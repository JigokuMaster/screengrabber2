based on [ScreenGrabber 4.2.0 by Nokia](https://github.com/SymbianSource/oss.FCL.sf.tools.rndtools/tree/master/screengrabber)

ScreenGrabber2 is an application for recording the screen or taking screenshoots.
it supports varoius formats including MP4 video.
it also supports taking shots from secondary screen.

# Requirements

* Symbian S60v3, S60v5, S^3 (hacked devices)

* P.I.P.S v1.7 (for version with MP4 support)

# Installation

Download the binaries from [releases page](https://github.com/JigokuMaster/screengrabber2/releases) 

* ScreenGrabber2.sis (with MP4 format, PIPS required)

* ScreenGrabber2_lite.sis (no MP4 format, PIPS is not required)

# Notes:

* Don't expect accurate framerates. ScreenGrabber is not suitable for recording games.

* The version with MP4 support requires more than 20MB of free RAM.

# Libraries

ScreenGrabber2 uses the following libraries. Thanks a lot to the developers of these projects.

* [msf_gif](https://github.com/notnullnotvoid/msf_gif) for exporting the recording as high-quality GIF Animation.
* [x264](https://github.com/mirror/x264) for x264 Encoder.
* [l-smash v2.14.5](https://github.com/l-smash/l-smash) for MP4 exporter.
